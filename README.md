
## free-form

帮助你处理表单状态,校验,简单的组合方式

## ✨ Features

- 常用组件校验

## 简介

非受控表单,单项数据流

form对象,可以操作单个Item状态
```
<FreeForm>
    <FormItem name="xx" label="userName">
        <input/>
    </FormItem>
</FreeForm>

```

### Installing

```
yarn add free-from
```

## Running the tests
```
yarn run test
```
